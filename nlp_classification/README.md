# Classifiers and Token Bias for ALA

This folder contains **lightweight text classifiers** and **token-importance (β) files** used by Adaptive Logit Adjustment.

You can either:

- Use the **included checkpoints** (for LLaVA / PaliGemma, and race models), and/or  
- Re-train models (especially for Qwen) using the provided training scripts.

---

## What is included

After cloning this repo you should see (non-exhaustive):

- `model.py` – `SimpleTransformerClassifier` used in the training scripts.
- Qwen training scripts:
  - `gender_classification_qwen.py`
  - `toxicity_classification_qwen.py`
  - `stereoset_classification_qwen.py`
- Pretrained gender models:
  - `gender_model_llava_pytorch_generated/`
  - `gender_model_paligemma_pytorch_generated/`
- Pretrained race models:
  - `race_model_llava_pytorch_generated/`
  - `race_model_paligemma_pytorch_generated/`
  - `race_model_qwen_pytorch_generated/`
- Pretrained toxicity models (for LLaVA / PaliGemma):
  - `toxicity_model_llava_pytorch/`
  - `toxicity_model_paligemma_pytorch/`
- Token-importance JSONs:
  - `importance_dict_llava_pytorch_generated.json`
  - `importance_dict_paligemma_pytorch_generated.json`
  - `importance_race_dict_llava_pytorch_generated.json`
  - `importance_race_dict_paligemma_pytorch_generated.json`
  - `importance_race_dict_qwen_pytorch_generated.json`
  - `importance_toxicity_dict_llava_pytorch.json`
  - `importance_toxicity_dict_paligemma_pytorch.json`

Qwen-specific token-importance files (e.g. `importance_dict_qwen_pytorch_generated.json`, `importance_toxicity_dict_qwen_pytorch.json`) are **not committed** and should be generated by running the training scripts below.

## Training Qwen classifiers and β files

All scripts assume the **same tokenizer** as the VLM is used.

### Gender (Qwen)

```bash
cd nlp_classification
python gender_classification_qwen.py --model qwen --gpu_id 0
```

This will create:

- `gender_model_qwen_pytorch_generated/pytorch_model.bin`
- `importance_dict_qwen_pytorch_generated.json`

### Toxicity (Qwen)

```bash
cd nlp_classification
python toxicity_classification_qwen.py --model qwen --gpu_id 0
```

This will create:

- `toxicity_model_qwen_pytorch/pytorch_model.bin`
- `importance_toxicity_dict_qwen_pytorch.json`

### Race (Qwen, Stereoset)

```bash
cd nlp_classification
python stereoset_classification_qwen.py --model qwen --gpu_id 0
```

This trains a race classifier and produces:

- `race_model_qwen_pytorch_generated/pytorch_model.bin`
- `importance_race_dict_qwen_pytorch_generated.json`

The ALA code will automatically pick up these files as long as they are under `nlp_classification/` (or the directory pointed to by `ALA_NLP_CLASSIFICATION_DIR`).

## Notes

- Checkpoints here are **small** (few MB) and suitable for including in the public repository.
- Image classifiers for FACET / Counterfactual tasks are simple Logistic Regression models; they are trained on the fly from image embeddings (`embedding/fairface_*_train.pt`) if the corresponding `image_*_classifier_*.pkl` file is missing.
- If you want to customize classifiers (different datasets or labels), you can fork these scripts and adjust the Hugging Face dataset or CSV loaders and the label mapping.
